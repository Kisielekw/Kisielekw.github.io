About

ColdLine Lapland is a game created by me and 2 of my teammates for the Thee Thing Game Jam organised by the University of Hull
In Three Thing Game, you receive 3 random words around one theme and are tasked to create a game involving these words within a weekend My team ended up with the words: Carollers, Lights, and Mincepie

The game is a twin-stick shooter in which the player shoots carollers using a mincepie shotgun as they try and stop you by singing Christmas carols at you.

The game was created in Unity and was a great learning experience for me as it was my first time using Unity and C# to create a completely playable game. I learned a lot about game development and the process of creating a game in a short amount of time.

Gameplay

The game is inspired by the classic twin-stick shooter game Hotline Miami. As you can probably tell we not only copied some of the mechanics but also the name of the game.
In the game, the player traverses the map shooting down enemy caroller using their trusty mincepie shotgun. The carollers will try and shoot you down using their deadly Christmas carols. At the end of the level the player will find the final boss, a giant bundle of tangled christmas lights which rapidly shoots out bundles of it's self at the player. Throughout the level, the player will find ammo crates to refill their supply of ammo.
This game loop provides a fast-paced game with a lot of action and a lot of fun. Due to the time constraints of the game jam, we could not add many of the features we wanted, such as additional levels and better enemy AI but we are proud of the game we created.
